# 📝 To-Do List App

A simple and clean To-Do List web application to manage daily tasks efficiently.

## 🚀 Features

* Add new tasks
* Mark tasks as completed (click)
* Delete tasks (right-click)
* Tasks saved using Local Storage
* Simple and responsive UI

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Local Storage

## 📂 Project Structure

```
index.html
style.css
script.js
```

## ▶️ How to Use

1. Enter a task in the input box
2. Click **Add Task** button
3. Click on task to mark as completed
4. Right-click on task to delete
5. Tasks are saved automatically

## 🌟 Future Improvements

* Edit task option
* Dark mode
* Due date feature
* Drag and drop tasks
